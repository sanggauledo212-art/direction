Materi

Teori

Materi ini mengenalkan kosakata arah dalam bahasa Arab melalui teks, suara, gesture, dan gerakan karakter Zayd di ruang 3D.

Kelompok 1: Arah Posisi أَمَامَ، خَلْفَ، يَمِين، يَسَار، فَوْقَ، تَحْتَ، دَاخِلَ، خَارِجَ
Kelompok 2: Petunjuk Berjalan تَقَدَّمْ، اِرْجِعْ، اِنْعَطِفْ يَمِينًا، اِنْعَطِفْ يَسَارًا، اِصْعَدْ، اِنْزِلْ
  1. Pilih menu 8 Arah untuk memahami posisi dasar.
  2. Pilih menu Petunjuk Berjalan untuk melatih instruksi bergerak.
  3. Aktifkan kamera, lalu pilih mode gesture yang sesuai. Model gesture sudah diganti dari swipe menjadi pose statis yang ditahan sebentar.
Model Zayd File GLB sudah tersedia di public/models/zayd.glb. Model ini dapat diganti kapan saja dengan model humanoid gratis lain.
